Git Quiz

What is Git and why is it so important?

Git is a version control system that is widely used in the programming world.Git is important because it provides cheap local branching, convenient staging areas, and multiple workflows It is used for tracking changes in the source code during software development and Git enables multiple developers or teams to work separately without having an impact on the work of others.

What’s a branch and why do we create them?

The git branch command creates, lists and deletes branches. A new branch is created to encapsulate the changes when you want to fix bugs or add new features.

What’s the working tree?

Working tree is your file system directory (including files and subdirectories) associated with a git repository which is filled with the files you adjust, where you insert new files and from which you delete the unnecessary files .

What’s the staging area?

Is a file, generally contained in your Git directory, that stores information about what will go into your next commit.

What’s the command to retrieve the latest version of your files from the remote serve?

Git fetch is used to retrieve the latest version of your files from the remote server.
